# Hanoi
Tower of Hanoi in C# by Antonia Hruschka (cc241062)

# About the Project
Tower of Hanoi is a mathematical Puzzle consisting of three rods and n number of disks. 
It has three rules:
Only one disk can be moved at a time.
Each move consists of taking one disk from the top of a stack and placing it on another. 
Disks can only be placed on disks larger than themselves. 

The Tower of Hanoi puzzle has a solution of a minimum of 2^n-1 moves, n being the number of disks.

This project solves the problem built with C# and .NET 10.0 SDK, as well as the computer science concepts of recursion and iteration. 

# Getting started
Before running the program, make sure you have .NET 10.0 SDK installed. 
If you're reading this, you should have all the necessary files needed to run this program, HOWEVER you can also clone the repository using this link: https://github.com/tonihru/Hanoi.git.

Open Hanoi.cs and the terminal, and run the program using one of the following commands:

1. Recursive approach:
dotnet run -recursive <number of disks>

2. Iterative approach:
dotnet run -iterative <number of disks>

# Roadmap
This Program contains, as previously mentioned, a recursive and an iterative solution to the Tower of Hanoi puzzle. The iterative solution is also visualized in the console using ASCII art (to the best of my ability.).
